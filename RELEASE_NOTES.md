## 📦 安装方式喵~

### 方式一：加载已解压的扩展喵~
1. 下载 `NyaTranslate-v2.0.0.zip`
2. 解压到本地文件夹
3. Chrome浏览器打开 `chrome://extensions/`
4. 开启右上角"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择解压后的文件夹

### 方式二：开发者模式喵~
```bash
# 克隆仓库
git clone https://github.com/your-username/NyaTranslate.git

# 进入目录
cd NyaTranslate

# 在Chrome中加载文件夹
# chrome://extensions/ -> 加载已解压的扩展程序
```

## ⚙️ 配置说明喵~

### API Key 配置喵~
1. 打开扩展设置页面
2. 在"API配置喵~"页面填入以下任一API Key：
   - **DeepSeek**: 从 [platform.deepseek.com](https://platform.deepseek.com/api_keys) 获取
   - **通义千问**: 从 [阿里云百炼平台](https://bailian.console.aliyun.com/) 获取
3. 点击"保存API配置喵~"

### 触发规则喵~
- **显示图标喵~**: 划词后出现小气泡图标
- **直接搜索喵~**: 划词后直接弹出面板
- **双击搜索喵~**: 双击选词时直接搜索
- **组合键触发喵~**: 按住修饰键划词强制搜索

## ⚠️ 注意事项喵~

- 需要 **Chrome 88+** 或基于Chromium的浏览器喵~
- 需要配置至少一个 **API Key** 才能使用喵~
- API Key仅保存在本地，绝不上传喵~
- 每次查询的文本直接发往你配置的API端点喵~
